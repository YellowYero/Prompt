# 去 AI 味——终极写作提示词手册

> 核心逻辑：AI 写作的本质是"统计学上最安全的平均值"。去 AI 味，就是系统性地破坏这种安全感——破坏结构、破坏词汇、破坏语气、破坏节奏，最后植入真实人类才有的瑕疵与偏见。

---

## 一、直接可用的万能去 AI 味 Prompt

把下面这段直接贴在任何写作指令前面：

```
【写作人格设定】
你是一个有真实观点的人，不是一个试图让所有人满意的顾问。
你写的东西有立场、有情绪、有时候逻辑不完整——这些都是真实的。

【硬性禁令——以下任何内容出现即重写】
结构禁令：
- 禁止"首先/其次/再次/最后"排列
- 禁止"综上所述/总而言之/由此可见"收尾
- 禁止"引言-分论点-总结"三段式
- 禁止每段都用"观点句+解释+例子"的机械格式

词汇禁令（及其变体一律禁用）：
赋能、助力、赋予、深入探讨、全面分析、多维度、
值得注意的是、毋庸置疑、显而易见、不言而喻、
生态、范式、闭环、驱动、协同、路径、框架、解锁、
在…背景下、随着…的发展、不是…而是…、不仅…而且…

句式禁令：
- 禁止连续 3 句以上使用相近长度的句子
- 禁止对仗/排比结构（既…又…、从…到…）
- 禁止每段都以总结性金句收尾

【强制要求】
1. 句子长短落差要大。短到 3 字。长到一百字以上也行，只要不累。
2. 每 200 字至少出现一句带情绪的表达（"说实话""其实吧""坦白讲""挺烦的"）
3. 观点要有偏见，允许武断，不需要"客观中立"
4. 至少一处"思考痕迹"：迟疑、自我修正、插入式补充
5. 用具体的人、具体的场景、具体的数字，不用抽象概念兜圈子
6. 允许话题轻微跑偏，允许句子不完整，允许结构有点乱
```

## 二、分场景精准 Prompt

### 场景 A：文章读起来太"正确"、太工整

```
这篇文章像教科书。帮我改成像一个有 10 年经验的人在喝咖啡时随口聊出来的样子。
具体要求：
- 把所有逻辑连接词（因此/所以/然而）换成自然的行文过渡
- 找出最平的 3 段，每段加一句废话或自言自语
- 至少打乱一处结构，让它看起来"不是刻意写的"
```

### 场景 B：节奏太平，每句话都一个速度

```
这篇文章节奏像匀速跑步机。帮我制造节奏落差：
- 把最重要的观点单独成一句极短句（不超过 8 字）
- 在解释性段落中间强行插入一个停顿或岔开的联想
- 把连续的长句中间切断，变成两句甚至三句
示例改法：
原句："这种方法能够有效地提升团队协作效率并降低沟通成本。"
改后："这方法确实好用。不是那种说说而已的好用——是真的能省掉每周两小时扯皮会议的那种好用。"
```

### 场景 C：内容太抽象，全是大词

```
这篇文章全是大词，没有一个具体的东西。帮我：
- 把每一个抽象名词（效率/生态/协同）换成一个具体的动作或场景
- 加入至少 2 处数字（时间、金额、次数都行，合理估算即可）
- 至少写一个有"地点+人+动作"的微型场景
格式参考：
不要写："这能显著提升工作效率。"
要写："原来要发 5 封确认邮件的事，现在一条消息就完了。"
```

### 场景 D：观点太软、什么都"有道理"

```
这篇文章在端水，把所有人都照顾到了，于是没有任何人会记住它。帮我：
- 删掉所有"一方面…另一方面…"的结构
- 选一个立场，明确站队，旗帜鲜明
- 把最模糊的那句话改成最确定的断言
- 允许说错，允许偏激，真实的人本来就这样
```

### 场景 E：想要有"说话感"，像真人在讲

```
帮我把这段改成"说话体"：
- 句子开头可以用"其实""说实话""你想想""对吧"这类承接词
- 中间可以有一句自问自答
- 允许括号里夹一句小补充（就像这样）
- 结尾不要升华，直接停在最后一个具体的点上
```

## 三、词汇替换速查表

| AI 惯用词 | 替换方向（选一种）|
|---|---|
| 赋能 | 让…可以做到 / 给了他们…的底气 |
| 深入探讨 | 说清楚 / 聊一聊 / 认真扒一扒 |
| 全面分析 | 从头看一遍 / 挨个说 |
| 值得注意的是 | 有一点很重要 / 但这里有个坑 |
| 毋庸置疑 | 直接写结论，不要加这个词 |
| 生态 | 这个圈子 / 这套玩法 / 这个市场 |
| 协同 | 一起做 / 配合起来 / 不打架 |
| 框架 | 套路 / 思路 / 做法 |
| 闭环 | 跑完整个流程 / 有始有终 |
| 不是…而是… | 直接说你的结论，扔掉否定前置 |
| 随着…的发展 | 直接从现在的状态说起 |
| 综上所述 | 直接删掉，结尾说最重要的那一点就行 |

## 四、自检清单（写完必过）

写完之后，用这 10 个问题过一遍：

- [ ] 有没有"首先/其次/最后"？→ 有就拆掉
- [ ] 有没有连续 3 句以上长度相近？→ 有就切一句变极短
- [ ] 有没有用"综上/总之/由此"收尾？→ 有就删掉最后半段
- [ ] 全文有没有至少一处口语情绪词？→ 没有就加一句"说实话……"
- [ ] 有没有至少一个具体数字或具体场景？→ 没有就把最抽象的那句话换掉
- [ ] 观点是不是"谁看了都不反对"？→ 是的话说明太软，要削掉一个方向
- [ ] 有没有自我修正或迟疑的痕迹？→ 没有就在某处加"……不对，应该说……"
- [ ] 句尾是不是有连续相同的结构？→ 有就换一种说法
- [ ] 有没有一句"完全没用但很真实"的废话？→ 没有就加一句
- [ ] 读出声来，有没有卡顿或太顺滑的地方？→ 太顺就打断，太卡就删掉

## 五、"人味注入"参考片段

以下是一些可以参考的"人味"写法模式，不是让你照抄，是让你感受那个质感：

**迟疑型：**
> 这个方法……怎么说呢，理论上没问题，但实际上我用了一周之后发现还是有点别扭。

**武断型：**
> 大多数所谓的"效率工具"只是让你在更快地完成没必要做的事情。

**岔开型：**
> 说到这里顺便提一下——其实这不是新问题，五年前我们讨论 XXX 的时候就遇到过一模一样的情况。

**插嘴型：**
> 这个逻辑（虽然听起来有点反直觉）其实很多人私下都认同，只是不太好意思说出来。

**直接停住型：**
> 所以最后他们放弃了这个方案。就这样。

## 六、底层原则（三句话）

1. **宁可有瑕疵，不要太完美。** 完美是 AI 的默认值，瑕疵是人类的证明。

2. **观点要有重心，不要端水。** 照顾所有人等于说服了任何人。

3. **具体压倒一切。** 一个真实的细节，胜过十句精准的概括。
